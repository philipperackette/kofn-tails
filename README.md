# Solution Open Source de partage de secret (Shamir) sur Tails

Ce dépôt contient :
- Une procédure HTML autonome sous Tails pour organiser une cérémonie de partage de secret k-sur-n (Shamir), avec génération et protection d’un secret maître (RSA 4096, Ed25519, AES-GCM).
- Un document LaTeX/PDF qui formalise mathématiquement le protocole et ses primitives (corps finis, Shamir, HKDF, PBKDF2, Ed25519, RSA-OAEP, AES-GCM).

Publics cibles :
- PME / équipes sécurité sans HSM
- Enseignants, agrégatifs, étudiants (CPGE, TIPE)
- Communauté crypto / sécurité pour audit et améliorations
